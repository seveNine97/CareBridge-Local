# CareBridge Local Submission Pack

This folder contains ready-to-edit submission assets for Kaggle Gemma 4 Good Hackathon.

## Contents

- `kaggle-writeup-template.md`: long-form technical and impact writeup template.
- `video-script.md`: 3-scene demo script with timing.
- `media-checklist.md`: assets checklist for media gallery.
- `judging-mapping.md`: explicit mapping to Impact, Storytelling, and Technical criteria.
- `demo-runbook.md`: exact commands and fallback flow for live demo day.
- `release-notes-v1.md`: concise release summary for repository and website.

## How to use

1. Fill placeholders marked with `TODO`.
2. Record the demo following `video-script.md`.
3. Capture screenshots and attach according to `media-checklist.md`.
4. Paste the final writeup into Kaggle and link repo + video + live site.
